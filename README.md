# ILDiligence
勤之时 - 基于番茄工作法与静心白噪音， 保持专注，勤奋，劳逸结合的任务清单工具

如何运行：
 - 下载或者Fork，请耐心等待，因为有几个音乐文件，整个工程大概16M。
 - 然后执行 pod install
 - 打开workspace文件编译运行即可。
 - 如需要分享等功能，请自行去ILKeysConstants.m 文件配置ShareSDK，QQ，Weixin的ID等值。
